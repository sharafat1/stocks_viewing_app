# stocks_viewing_app
This Stocks Viewing Application was built as a part of the CAB230 - WebComputing subject at QUT. This repository contains the front-end application and the back-end API for the Stocks Viewing Application which was the requirements of CAB230 Assignment 1 and 2.

# Application Structure
Inside this repository there are two folders one contains the front-end and the other contains the back-end. Inside both folders there are README.md files which will guide you through the compilation of this application.
