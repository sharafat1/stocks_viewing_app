# Running this React Application

This is CAB230 Assignmnet 1 - Client Side

To install/to run the project follow the steps below:

Step 1: Download/Clone the project folder and unzip it.

Step 2: Then go to the client directory using a command line, then type npm install, which will install all the required packages and libraries the application needs to function.

Step 3: Then to run the project go to the client directory, in the command line you need to type the following command: npm start

Step 4: Then go to the http://localhost:3000 to test the application.
